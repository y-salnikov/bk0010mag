# bk0010mag
Android приложение для загрузки bin-файлов компьютеров семейства БК0010(11) через интерфейс магнитофона.

Программа написана на python с использованием фрэймворка kivy, предназначена для сборки черер buildozer.

По умолчанию, в место имени файла передаются пробелы, что позволяет не вводить имя файла на БК. Есть возможность редактировать имя, адрес загрузки и длину файла перед загрузкой.
Есть поддержка турбо-загрузчика, взятого из ~~онлайн~~ JavaScript конвертера файлов http://thesands.ru/bk0010/wav-converter/

Программа запрашивает специальное разрешение MANAGE_EXTERNAL_STORAGE для доступа к файлам на устройстве.
